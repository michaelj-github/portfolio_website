Another fun project showing JavaScript DOM manipulation and the use of Axios to make an API call.

Concentration Solitaire

This was originally a Springboard Software Engineering program project. I modified it to use the deckofcardsapi API to get a random selection of playing cards. CSS Flexbox makes it respond to screen size.

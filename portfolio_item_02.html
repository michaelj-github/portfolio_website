Project title: Art Explorer

a fun website for art lovers

Art Explorer is a fun website for art lovers to use to find and share images of works of art from The Metropolitan Museum of Art.

Art Explorer

This is one of the projects I did for the Springboard Software Engineering program. I built this app as a demonstration project using Python, PostgreSQL, Flask, and Bootstrap with the Met Museum API. It is deployed on Heroku. The code and documentation are available on GitHub including a complete test suite using Python unittest.

Tech Stack
  • Python
  • Flask
  • SQLAlchemy
  • WTForms
  • PostgresSQL
  • Bootstrap 5
  • Bcrypt for passwords
  • HTML/CSS/JavaScript for the front-end
  • Heroku with Gunicorn for deployment
